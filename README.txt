Passo a passo para a execução do sistema da etapa 1:

1 - Entre na pasta "Etapa-1"

2 - Instale o NodeJs: https://nodejs.org/pt;

3 - Clone o repositório;

4 - Ajuste os dados do banco de dados, como o user, password e database (De preferência com o nome "univali"), no arquivo db.js dentro da pasta src;

5 - Utilize o comando "npm install" no terminal para instalar as dependências necessárias;

6 - Utilize o comando "npm init -y" no terminal caso não possua o arquivo "package-lock.json";

7- Utilize o comando "npm run iniciarProjeto" para inicar o projeto;
